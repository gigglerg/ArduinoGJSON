Detailed Description

An class for a user specified buffer when JSON parsing.

This allows callers to specify where the JSON parser string I/O resides and allows for the situation where a user may want to reuse a UART buffer rather than create another.

Examples
simple_object_map.ino, simple_object_map_map.ino, string_replace_1.ino, string_replace_2.ino, string_search_1.ino, string_search_2.ino, string_search_3.ino, and string_to_object_to_string.ino.
