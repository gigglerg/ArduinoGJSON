Detailed Description

template<uint8_t LENGTH>
class gjson::JSONMapTupleArray< LENGTH >

A builder, helper class representing a JSON tuple (Map) between a string property/attribute and another Array.

Parser will not use this type of element in-memory.

This class inherits from JSON array for the property value and contains another string for the property/attribute name. It was important to keep this layout as JSONElement::Next() isn't virtual.

Template Parameters
LENGTHMaximum length (including NULL terminator) in characters of a string attribute/property
Examples
custom_object_parse.ino, object_search_1.ino, and object_search_2.ino.
