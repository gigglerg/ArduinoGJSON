Detailed Description

template<uint16_t BL>
class gjson::Buffer< BL >

An class for an internally managed character buffer when JSON parsing.

Use this in situations where you require some storage for JSON, when data may get changed and there is no user string to be reused.

Template Parameters
BLMaximum buffering length in characters (including any NULL terminator)
Examples
object_update_1.ino, object_update_2.ino, simple_object_array.ino, simple_object_array_map.ino, and simple_object_map_map_2.ino.
