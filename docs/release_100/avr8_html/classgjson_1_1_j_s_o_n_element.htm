Detailed Description

template<uint8_t LENGTH>
class gjson::JSONElement< LENGTH >

A class representing a basic JSON in-memory element.

This is a parent class to all possible JSON element types and JSON strings ultimately get parsed into objects based off this class

Template Parameters
LENGTHMaximum length (including NULL terminator) in characters of a string attribute/property
Examples
custom_object_parse.ino, simple_object_array.ino, string_to_object_1.ino, string_to_object_2.ino, and string_to_object_to_string.ino.
